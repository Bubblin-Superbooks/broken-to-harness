There was no need of speech; it was all known, all settled, all arranged; that restored glove, that one fervent sentence, that one look in which each seemed to read the secrets of the other's soul, had done it all. This was first love, undisturbed by the fact that on either side there had probably been some half-dozen attacks of that spurious article, that saccharine bliss, that state of pleasant torture which reveals itself in sheep-like glances and deep-drawn sighs, in a tendency to wear tight boots and to increase the already over-swollen tailor's bill, to groan and be poetical, and to shrink from butchers' meat. Although the existent state of Barbara and Churchill had none of these characteristics, it was still first love.
